Hormuz Shadow: Global Shipping Disruption Analysis

Project Overview

This project analyzes the impact of geopolitical disruption on global shipping routes, focusing on rerouting scenarios when access to the Strait of Hormuz is restricted.

The study combines data analysis, simulation modeling, statistical validation, and geospatial visualization to quantify the effects on shipping cost and travel time.



Objectives

* Analyze how rerouting affects shipping distance and time
* Estimate increase in fuel cost due to longer routes
* Validate impact using statistical testing
* Visualize route changes on a global map

Tools & Technologies

* Python
* Pandas (data processing)
* Matplotlib (visualization)
* SciPy (hypothesis testing)
* Statsmodels (regression analysis)
* Folium (geospatial mapping)

Methodology

1. Data Preparation

* Cleaned and structured AIS, fuel, and port datasets
* Selected relevant features (distance, time, ship ID)

2. Simulation Modeling

* Created a “Before” (normal route) and “After” (rerouted) scenario
* Increased distance and travel time to simulate disruption

3. Visualization

* Compared cost and time using bar charts
* Plotted distance vs fuel cost using scatter plots

4. Statistical Analysis

* Conducted T-tests to validate cost and time increases
* Built a linear regression model to analyze cost vs distance relationship

5. Geospatial Mapping

* Visualized shipping routes using interactive maps
* Compared normal vs rerouted paths

Key Findings

* Rerouting significantly increases shipping distance and travel time
* Fuel cost rises proportionally with distance
* Statistical testing confirms the changes are significant (p < 0.05)
* Geospatial visualization clearly shows route deviations

Key Concepts Applied

* Feature engineering

* Linear regression
* Simulation modeling
